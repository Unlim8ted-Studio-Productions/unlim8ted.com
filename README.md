# Unlim8ted.com

![Website Status](https://img.shields.io/website?down_color=red&down_message=offline&up_color=green&up_message=online&url=https%3A%2F%2Funlim8ted.com)

This repository contains the website source, supporting service assets, media catalogs, and internal maintenance scripts for [unlim8ted.com](https://unlim8ted.com).

## Purpose

The codebase supports:

- Public website pages and product/category landing pages
- Browser-based interactive experiences
- Supporting assets for catalog, portfolio, and media presentation
- Internal scripts for publishing, asset management, merchant feeds, and product data maintenance
- Prototype and presentation materials for selected software, AI, and hardware projects

## Repository Layout

### `frontend/`

Primary website source. This directory includes:

- Core site pages such as `index.html`, `404.html`, `about/`, `contact/`, `portfolio/`, `privacy/`, and `tos/`
- Shared client-side components in `components/`
- Product and category pages in `products/`
- Site-level files such as `robots.txt`, `sitemap.xml`, and `google_merchant_products.xml`

### `frontend/products/`

Product and category content grouped by area, including:

- `ai/`
- `books/`
- `films/`
- `games/`
- `hardware/`
- `images/`
- `music/`
- `physical-items/`
- `podcasts/`
- `software/`

### `services/`

Supporting service-side assets used by product integrations. The current repository includes `services/wisesize/`.

### `assets/`

Shared media and product imagery used across the site.

### `tools/`

Internal scripts and datasets used for maintenance and operations, including:

- Product page generation and refactoring
- Asset-path and catalog updates
- Merchant feed and sitemap generation
- Ecommerce and print-related data workflows
- AI-related datasets, experiments, and support utilities

## Notable Areas

The repository includes several larger project sections:

- `frontend/products/ai/meatball/` for the Meatball AI browser experience
- `frontend/products/software/wise-size/` for Wise Size product pages
- `frontend/products/hardware/unlim8ted-phone/` for Unlim8ted Phone presentation and embedded UI preview files
- `services/wisesize/` for Wise Size integration assets
- `tools/meatball ai/` for AI-related scripts and datasets
