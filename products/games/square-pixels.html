Introduction

Welcome to our pixelated sandbox adventure! Inspired by Terraria, Square Pixel blends exploration, crafting, and action-packed survival in a procedurally generated world with endless possibilities.

Features

  • Expansive World: Explore diverse biomes, dungeons, and hidden treasures.
  • Craft and Construct: Gather resources and build your dream structures.
  • Monstrous Challenges: Fight creatures and bosses as you progress.
  • Multiplayer Magic: Play together locally or online (planned).
  • Modder’s Paradise: Robust modding support (planned).
  • Day and Night: Shifting conditions that change encounters and strategy.

Getting Started

  1. Clone the repository:

    git clone https://github.com/Unlim8ted-Studio-Productions/Square-Pixels.git

  2. Install dependencies:

    pip install pygame pillow pymovie pyperclip playfab captcha astor

  3. Launch the game script or executable.

You can also download the latest executable from the releases page. For help with builds, contact support@unlim8ted.com.

How to Play

Controls:

  • Move with WASD or arrow keys.
  • Open inventory with E.
  • Right-click in inventory to create an item.
  • Left-click to place an item in a slot.
  • Left-click on blocks to destroy them.

Objective:

  1. Create a new character.
  2. Explore, gather resources, and craft items.
  3. Build structures to protect yourself and store loot.
  4. Defend against monsters and bosses.
  5. Survive and master the world.
