How sizing works

WiseSize matches a user profile (body measurements + fit preference) against your product’s sizing chart. We support two chart types:
  • Body charts: each size contains measurement ranges [min,max].
  • Garment charts: store flat-lay garment measures; we still store them as [min,max].

How to measure garments

Recommended measurements:
  • Top: chest, waist, shoulder, length
  • Bottom: waist, hip, inseam, rise
  • Dress: bust, waist, hip, length
Use consistent units (cm recommended). For garment measurements, measure flat on a table with a tape.

What gets published

Only items marked published are used by live embeds. Submitted items go through verification first. Rejected items require edits/resubmission.

How to embed

Add this script to your product page. Replace values with your merchantId and itemId.
<script src="https://services.unlim8ted.com/wisesize/v1/embed.js" data-wisesize-merchant="YOUR_MERCHANT_ID" data-wisesize-item="YOUR_ITEM_ID" defer></script>
You can copy per-item embed snippets from the Products page.

FAQ + troubleshooting

  • “Not found yet” after DNS verification: wait for DNS propagation and try again.
  • HTTP verification fails: ensure the file is public and returns 200 at the exact path.
  • Dashboard keeps redirecting to setup: your merchant doc must have websiteVerified=true.
  • Permission errors: confirm you’re writing under /wise-size/app/... and logged in.
